# Harbor Task – JSON Filter Task

## Overview

This project implements a **Harbor task** that processes JSON data.
The agent reads an input file, filters users based on age, and writes the result to an output file.

This task was created as part of the **Harbor New Hire Assignment**.

---

## Task Description

The agent must:

1. Read user data from `/app/input.json`
2. Filter users whose **age is greater than or equal to 18**
3. Write the **names of the filtered users** to `/app/output.txt`
4. Each name should appear on a **new line**

---

## Example

### Input (`input.json`)

```json
[
  {"name": "Alice", "age": 25},
  {"name": "Bob", "age": 14},
  {"name": "Charlie", "age": 30},
  {"name": "Diana", "age": 17},
  {"name": "Eve", "age": 22}
]
```

### Output (`output.txt`)

```
Alice
Charlie
Eve
```
